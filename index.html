Hi, I'm Scott.

I'm a front end developer and web accessibility advocate at Shopify.

Some history

I've been coding on the web for a long while.

I first started out learning HTML 4.0 and table layout.

A few years later I moved over to learn back end with PHP and MySQL. Around 2009-ish I realized my love for front end when the agency I was with at the time didn't have anyone available to work on a project.

It was about 2011 when I first experienced screen reader technology. Ever since then my primary focus has been to create accessible user experiences with HTML, CSS, and JavaScript.

I also…

  • co-organize the #CodePenOttawa meetup;
  • co-organize the A11Y Tour conference;
  • co-organize the Canada Learning Code YGK meetup;
  • co-run the A11Y Store;
  • co-lead the Canada Learning Code Kingston chapter;
  • create and lead accessibility workshops at HackerYou;
  • maintain The A11Y Project.
